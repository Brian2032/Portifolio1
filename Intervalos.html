Intervalos Reais

Os intervalos reais são diversos conjuntos de números baseados em condições pré-estabelecidas. Podem ser representados de três formas diferentes: uma reta numérica, uma escrita matemática como (a, b] ou uma escrita matemática como {x ∈ ℝ | a < x < b}.
Para saber a diferença entre um elemento incluído / excluído dentro destas escritas, você deve observar o símbolo que representa esse elemento. Na reta numérica, o elemento que está incluído é representado por um círculo preenchido, já o elemento excluído é representado por um círculo não preenchido. As outras duas maneiras de escrever o intervalo possuem a mesma regra com pouca diferenciação, elas utilizam '<' ou '>' para elementos incluídos e '≤' ou '≥' para elementos excluídos. Os elementos incluídos são chamados de intervalo fechado e os excluídos de intervalo aberto. No caso da maneira de escrever (a, b], o parenteses representa um intervalo aberto e o colchetes representa um intervalo fechado.
Exemplos de como fazer representações de intervalos reais:

Reta numérica (a, b] {x ∈ ℝ | a < x < b}
Os números do 15 (incluído) ao 19 (incluído) __15•⋯⋯⋯19•__ [15, 19] {x ∈ ℝ | 15 ≤ x ≤ 19}
Os números do 15 (excluído) ao 19 (excluído) __15ο⋯⋯⋯19ο__ (15, 19) {x ∈ ℝ | 15 < x < 19}
Os números do -∞ ao 50 (incluído) __-∞ο⋯⋯⋯50•__ (-∞, 50] {x ∈ ℝ | -∞ < x ≤ 50}
Os números do -12 (incluído) ao -2,1 (excluído) __-12•⋯⋯⋯-2,1ο__ [-12, -2,1) {x ∈ ℝ | -12 ≤ x < -2,1}
Os números do -⅝(incluído) ao ∞ __-⅝•⋯⋯⋯∞ο__ [-⅝, ∞) {x ∈ ℝ | -⅝ ≤ x < ∞}
