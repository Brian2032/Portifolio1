Símbolos Matemáticos e Significados

X+ : significa que o conjunto não inclui números negativos.
X- : significa que o conjunto não inclui números positivos.
X* : significa que o conjunto não inclui o número zero.
X = { } : o que está dentro das chaves pertence ao conjunto.
X = { 1, ...} : as reticências representam que existem mais valores, antes, entre ou após valores.
⊂ : contém.
⊃ : está contido em.
⊄ : não contém.
⊅ : não está contido em.
⋃ : união com.
⋂ : intersecção com.

Os símbolos acima são utilizados quando compararem dois conjuntos.

∈ : pertence a.
∉ : não pertence a.

Os símbolos acima são utilizados quando compararem um valor com um conjunto.

| : tal que.

Símbolo extra acima.

• intervalo fechado.
ο intervalo aberto.
⋯ linha pertencente ao intervalo
_ linha não pertencente ao intervalo
() : intervalo aberto.
[] : intervalo fechado.

Os símbolos acima são escolhas para a representação de meu trabalho na parte de intervalos reais. Em geral, um intervalo aberto é representado pelo contorno de um círculo, o intervalo fechado é representado pela área de um círculo e a reta mais destacada representa o intervalo. Os colchetes sempre representam um intervalo fechado e os parenteses sempre representam os abertos.
